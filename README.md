# Кофейня «Уголок» — прототип на Next.js

## Запуск

```bash
npm install
npm run dev
```

Открой в браузере: http://localhost:3000

## Страницы

- `/` — главная (название, слоган, кнопка в меню, промо)
- `/menu` — меню (напитки и блюда, по кнопкам — отдельно списки)
- `/menu/[id]` — страница блюда (название, описание, цена, «Назад в меню»)
- `/about` — о нас
- `/contacts` — где нас найти (адрес, часы, контакты, заглушка карты)

## Сохранение в Git

```bash
git init
git add .
git commit -m "Прототип кофейни"
```
